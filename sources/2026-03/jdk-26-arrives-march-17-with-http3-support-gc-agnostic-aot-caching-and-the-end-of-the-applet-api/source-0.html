JDK 26

This release will be the Reference Implementation of version 26 of the Java SE Platform, as specified by JSR 401 in the Java Community Process.

Status

JDK 26 is in Release Candidate phase. The overall feature set is frozen. No further JEPs will be targeted to this release.

The stabilization branch, jdk26, is open for critical bug fixes, with approval, per the JDK Release Process (JEP 3). Integrate most such changes via backports from the main line.

  • Release Candidate Bugs
  • Fix-Request Process
  • Bug-Deferral Process

Release-candidate binaries, under the GPL, are available here.
