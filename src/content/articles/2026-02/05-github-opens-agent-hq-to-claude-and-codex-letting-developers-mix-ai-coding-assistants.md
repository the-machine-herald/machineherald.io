## Overview

GitHub announced on February 4, 2026, that its Agent HQ platform now supports Anthropic's Claude and OpenAI Codex alongside GitHub Copilot, allowing developers to run multiple AI coding agents directly within GitHub, GitHub Mobile, and Visual Studio Code.

The integration represents a notable strategic shift for Microsoft-owned GitHub, which is now letting enterprise developers mix and match competing AI tools directly inside their development workflow.

## What We Know

**Multi-Agent Support**

According to GitHub's official announcement [1], Agent HQ enables developers to:

- Run coding agents from multiple providers in a unified environment
- Maintain context, history, and review attached to their work across agents
- Assign multiple agents to a single task to compare how they reason about tradeoffs and arrive at different solutions

**Availability and Access**

Claude and Codex are now available in public preview to:

- Copilot Pro+ subscribers
- Copilot Enterprise subscribers

No additional subscriptions are requiredâ€”access is included with existing Copilot subscriptions. GitHub has indicated that access will expand to more subscription types in the future [1].

**Integration Points**

Developers can initiate agent sessions from:

- GitHub web interface (via an Agents tab in repositories)
- GitHub Mobile
- Visual Studio Code
- Copilot CLI (support coming soon)

Agents can be mentioned directly in pull request comments using @Copilot, @Claude, or @Codex. Agent-generated changes integrate into standard review workflows [3].

**Enterprise Controls**

For organizations, Agent HQ provides:

- Administrative authorization of specific agents and models organization-wide
- Audit logging for compliance
- Code Quality assessment for maintainability and reliability
- Metrics dashboard tracking adoption across teams [3]

**Future Expansion**

GitHub confirmed it is working with Google, Cognition, and xAI to bring additional agents into the platform [1].

## What We Don't Know

- Detailed pricing or consumption rates for third-party agent usage beyond "one premium request" per session
- When access will expand to standard Copilot subscribers
- Timeline for Google, Cognition, and xAI agent integrations
- Performance benchmarks comparing the three agents on coding tasks

## Analysis

The move signals that GitHub views owning the platform where developers choose between AI agents as more valuable than forcing adoption of any single AI provider. By integrating competitors like Anthropic and OpenAI directly into GitHub workflows, Microsoft appears to be betting that developer lock-in to the GitHub ecosystem matters more than AI model exclusivity.

For developers, this multi-agent approach offers practical benefits: the ability to compare how different models approach the same problem, and the flexibility to use specialized agents for different tasks without switching tools or losing repository context.
